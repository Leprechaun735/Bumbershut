# Bumbershut
"All in 1 fun, under the sun"

Class Project Made with Tyson Bickett and Mitch Wilde
Later taken upon by Daydream Industries

Goal was to create an umbrella with solar panels to be able to charge items on the beach
  We also created a built in speaker system as well as battery storage for the extra charge

Won Purdue Polytec Design and Innovation People's Choice Award resulting in a $500 check
